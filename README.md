## Facts

- Basiert kontinuierlich auf allen aktuell verfügbaren Kurskalendern der DHBW Mannheim.
- Wird täglich um 06:00 Uhr Morgens aktualisiert.
- Zeigt die Raumbelegung der nächsten 3 Monate.
